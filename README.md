# ALU System Engineering - DevOps

System engineering and DevOps projects for the ALU Software Engineering program.

## ssh

Scripts related to SSH configuration and usage.

### 0-use_a_private_key

Connects via SSH to a server as the `ubuntu` user, using the private key
`~/.ssh/school`.

Usage:

```
./0-use_a_private_key <server_ip_or_hostname>
```
